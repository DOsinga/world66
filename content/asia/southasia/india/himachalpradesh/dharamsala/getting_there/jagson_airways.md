It is possible to fly from Delhi to Kangra if need be. It's considerably faster than bus, car, or rail, but, it's also considerably more expensive. If memory serves, the flight is under 2hours. They don't fly everyday, so be sure to check the schedules. Actually, it (Kangra) used to be a part of nepal in the 17th century.
