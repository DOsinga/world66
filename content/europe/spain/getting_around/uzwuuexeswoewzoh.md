Windows Live Mail does indeed use the .eml famort. Even though Emailchemy doesn't have Windows Live Mail listed as a supported famort (yet), it can convert this to Thunderbird folders. Try this: using the conversion wizard, select RFC-2822 Message Folders in step 1, then select the Local Folders folder found inside the Windows Mail data folder. Convert this to Thunderbird mail folder and then copy the resulting files to the Thunderbird Mail/Local Folders folder. Email me if you need more help.
