---
title: "Getting Around"
type: section
---

Pereira's Megabus BRT system is the main public transit option. Modeled on Bogota's Transmilenio, it runs enclosed stations connected by fiber optics, so you can see when the next bus arrives. You will need a rechargeable smart card; a single fare is about 2,500 pesos. Taxis are plentiful and cheap, though you should confirm the meter is running before you set off.
