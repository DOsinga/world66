What it is

A travel guide that reads like a book, built like Wikipedia.

World66 started in 1998 as one of the first open-content travel guides on the web. All content lives in a folder of markdown files — no database, no CMS. Anyone with a GitHub account can contribute.

The site is served by a lightweight Django app that reads directly from the filesystem. It's fast, transparent, and easy to run locally.
