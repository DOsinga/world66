# Afrika Shrine

*Africa > Nigeria > Lagos > Nightlife > Africanshrine*

Run by Femi Kuti (son of Fela), this is a great rock temple. The best night to go there is sunday. When Femi is outside of the country the place can be rather empty, however, so you might want to check if he is around. The Shrine is located in a quiet street in Ikeja.
