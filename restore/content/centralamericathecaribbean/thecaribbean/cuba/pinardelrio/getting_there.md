# Getting There

*Centralamericathecaribbean > Thecaribbean > Cuba > Pinardelrio > Getting There*

If you want to go to Pinar del Rio, you will have to go via Havana. As for most places in Cuba getting somewhere is easiest if you hired your own car. If you didn't Pinar del Rio is reachable by bus, which leave quite regurarly from Havana. Another possibility is taking the train, but beware when I was there the train only went every other day, so plan in advance...
