# Budget flights to Helsinki and Tampere

*Europe > Finland > Gettingthere > Budget Flights To*

Use the European Low Cost Airline Guide search engine ( http://www.low-cost-airline.info) to find out which budget airlines fly into Finland airports (Mainly Tampere, some to Helsinki).

Often you can save many $/£/whatever by using these airlines instead of the main carriers.
